A) HTML Structure - top

Considering that Kronos is a page template html is rather robust, but it is divided into logical sections in order to make changes and maintenance easier. Every section begins with DIV of "section" class. ID attribute of this DIV is used for navigation purposes, and corresponds to a section name.

Header

Header contains the image logo. Image logo is located in "Kronos/images/logo" directory as logo.png file, and you can replace it with your own logo.

Menu is driven by java script and in order to change it, must bear in mind that every link has corresponding section DIV. If you change "href" attribute of a navigation link you mast also change the id attribute of his section DIV. For example if you wish to change first link in the navigation

<a href="#take-me-home" class="nav-active rounded"><span>Take Me Home</span></a>

into

<a href="#home" class="nav-active rounded"><span>Home</span></a>

you should also change its section DIV tag from

<div class="section first-section" id="take-me-home">

to

<div class="section first-section" id="home">

You can recognize section DIVs by the "section" class attribute, you can use this to navigate through code as every "section" DIV represents beginning of a section.

Gallery

Gallery is driven by the jQuery plug-in code4net.slideshow.js. Base element of the gallery is a gallery slide with dimensions 980x400px and it can contain anything you like as you can see in a demo.

Gallery can contain any number of slides, to insert a new slide just create new gallery slide DIV:

<div class="galery-slide">
</div>

Gallery navigation is created automatically, and you don't need to worry about it.

Main settings for gallery are located in "Kronos/js/main.js", for more information on them please refer to java script section of this document.

About

This is mostly textual part and you can replace text as you desire, please use "<p>" tag for separate paragraphs. Image is located in folder "Kronos/images/about/" and you can replace it with anything you like. For more information about images please refer to the PSD section of this document. If you want to change background color of the section, you can do it in "Kronos/css/site.css" (BACKGROUND logical section).

Kung fu

This section contains textual part, image and list of services. Every service contains icon, title of service and textual part. You can find image and icons in folder "Kronos/images/services/". For more information about images please refer to the PSD section of this document.

Showcase

Gallery of this section is driven by jquery.lightbox-0.5.pack.js plug-in. DIV (showcase-sub-container) contains nine DIVs (pic-frame).

<div class="pic-frame rounded">
   <div class="pic">
      <a href="images/portfolio/originals/portfolio_large_01.jpg"><span class="roll"></span><img src="images/portfolio/thumbs/portfolio_small_01.jpg"/></a>
   </div>
   <span>Terre estere doloremque sei perse</span>
   <p>Website / Print / Marketing</p>
</div>

Thumbnail pictures are located in "Kronos/images/portfolio/thumbs/" folder and full sized pictures are located in "Kronos/images/portfolio/originals".

Kronos Team

Images for this section are located in folder "Kronos/images/team/". Unordered list with socio icons is wrapped by <div class="socio"> tag. You can add or remove socio icons you wish. It is only mather that tag <li class="last"> be the last one.

News

This is mostly textual part and you shouldn't have trouble replacing text of it with your actual news. You can style buttons in BUTTON logical section of site.css file.

Contact Us

In this section you will find the contact form, textual parts and the part which contains socio icons. Every paragraph of text should be wrapped with "<p>" tag. You can insert socio icons you wish, there are quite a few icons shipping with Kronos theme, for more info on socio icons please see PSD section of this document.

B) CSS Files and Structure - top

All CSS flies for Kronos theme are located in folder "Kronos/css":

  • reset.css - contains specific styles used to reset all browser default styles, in order to avoid different rendering.
  • site.css - most part of the theme CSS is located here divided in logical sections
  • jquery.lightbox-0.5.css - CSS styles needed for jquery.lightbox-0.5.pack.js plug-in

If you have any additional question about how to customize some part on template, feel free to contact us.

C) JavaScript - top

All java script libraries are located in "Kronos/js/" folder:

  • jquery-1.7.2.min.js - jQuery is a JavaScript library that greatly reduces the amount of code that you must write
  • code4net.slideshow.js - is our jQuery plug-in and it manages Gallery slides
  • modernizr-2.6.2.min.js - Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the user's browser.
  • jquery.lightbox-0.5.pack.js - is jQuery plug-in and it manages pop-up gallery in showcase section
  • on.scroll.js - is our java script library, based on jQuery, and it's used to manage navigation and scroll effect on the page, also tracks scrolling of the page and updates navigation depending on section currently displayed.
  • main.js - is a java script file containing mainly initialization methods for theme scripts
  • ajax.js - is our java script library that enables asynchronous communication with contact.php scrip while sending contact email message.
  • pop-up-mask.js - is a java script file. It manages background of modal window with message depending of validating form results.
  • script.js - is a java script file used to select appropriate links and call the jQuery lightBox plugin.
  • PIE.js - is a javascript based plugin and it makes Internet Explorer 8 capable of rendering CSS3 border-radius decoration features.
Settings (Gallery)

Java script settings regarding this section can be found in "Kronos/js/main.js" file:


$('.galery').code4netslideshow({
   'navigation' : [
      {
         'id' : 'galery-nav',
         'type' : 'bullet'           //bullet/arrow
      }
],
   'slideInterval' : 5,
   'pauseInterval' : 10,
   'type' : 'horizontal',
   'autoSlide' : true
});

Settings you could change(add):

  • slideInterval - is interval between two slide changes, default 5(sec)
  • pauseInterval - is period in seconds before next slide change, after user has clicked navigation button, default 10(sec)
  • scrollSpeed - is the speed of the slide effect, values slow/fast, default slow
  • type - marks type of transition effect between slides, values horizontal/vertical/fade, default horizontal
  • autoSlide - if you wish for your slide show to changes slides automatically set this value to true, otherwise false, default is true.

F) PHP Code Explanation - top

IMPORTANT:

Regarding contact form as main part of Contact Us section, only thing you need to do is to setup your email address. To do this, please open "Kronos/contact.php" file and in a second line:

define('TO_EMAIL', '"John Doe" <john.doe@your-actual-domain.com>');

replace "John Doe" and "john.doe@your-actual-domain.com" with your valid info.

Note: Do not remove any quote or double quote signs while doing this.

One notice: this will be functional only on your server, php file can not be executed on your local computer.
